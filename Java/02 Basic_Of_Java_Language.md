### 数据类型

1. 强类型语言：严格符合规定，必须先定义后使用，安全性高，速度慢
2. 弱类型语言：JS

- 基本类型：
  - byte -128-127 1字节
  - short -32768 - 32767 2字节
  - int 4字节
  - long 8字节 30L
  - float 4字节，要专门在数据后标识F 50.1F
  - double 8字节
  - char 2字节
  - boolean 1字节
- 引用类型
  - 类
  - 接口
  - 数组

 **面试补充**

1. 0b 二进制 0开头八进制 0x十六进制
2. 浮点数拓展：舍入误差，最好完全避免使用浮点数进行比较 使用BigDecimal类
3. 字符拓展：字符转换为int，所有的字符本质都是数字Unicode  '\u0061'代表a
4. 转义字符 \t  制表符 \n 换行
5. 对于对象引用，调用==方法是否一致
6. 布尔值扩展 if(true)

### 类型转换

1. 不同类型数据转换为同一类型，然后进行运算
2. 优先级从低到高 byte,short,char ----- int ----- long ----- float ----- double
   - 强制类型转换 (类型) 变量名 从高到低 如int到byte
   - 自动类型转换  低到高自动

3. 不能对布尔值进行转换
4. 不能把对象类型转换为不相干类型
5. 转换的时候可能存在内存溢出等问题
6. JDK7新特性，数字之间可以加入下划线分割
7. 转换之前如果已经存在溢出问题，赋值语句也会出错 EX long x = a * b; a,b为int溢出，可以把一个数转换成long，就ok

### 变量

1. 变量必须有类型，既可以是基本类型，也可以是引用类型
2. 变量名必须是合法的标识符
3. 变量声明是一个完整语句需要分号结尾
4. 变量作用域
   1. 类变量 ：static
   2. 实例变量： 不加static，仍然属于对象的
   3. 局部变量：方法区里的

5. 除了基本类型初始化null

### 常量

1. 初始化后无法再被改变
2. final关键字修饰
3. 一般使用大写字符 PI = 3.14  final static这些是修饰符不区分顺序

### 命名规范

1. 所有变量，方法，类名见名知意
2. 类成员变量 首字符小写驼峰原则：monthSalary
3. 局部变量：首字母小写和驼峰原则
4. 常量：大写字母和下划线 MAX_VALUE
5. 类名：首字母大写和驼峰原则
6. 方法名：首字母小写和驼峰原则

### 运算符

1. 算术运算符： + - * / % ++ ==

2. 赋值运算符： =

3. 关系运算符： > < >= <= == != instanceof(面向对象)

4. 逻辑运算符：&& || ！

5. 位运算符 & | ^(异或)  ~(非) >> << >>>()

6. 条件运算符 ？：

7. 扩展赋值运算符： +=，-=，*=，/=

8. 运算操作有高优先级向上转型，默认int

9. ++a a++

10. Math.pow(2,3)次方操作

11. << *2 >>/2 效率极高的位运算

12. 注意优先级
