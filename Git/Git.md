# Git

## 1 什么是版本控制工具

**版本控制工具**： 用于实现**版本控制**这一思想的**工具**；这种**控制工具**大致分为集中式**版本控制工具**（CVS ;SVN ;VSS）和分布式**版本控制工具**(Git and so on)。

1. 实现跨区域多人协同开发
2. 追踪和记载一个或多个文件的历史记录
3. 组织和保护你的源代码和文档
4. 统计工作量
5. 并行开发，提高开发效率
6. 跟踪记录整个软件的开发过程
7. 减轻开发人员的负担，节省时间，同时降低人为错误
